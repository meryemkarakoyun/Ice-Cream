# 🍦 Ice Cream App 

A modern ice cream shop landing page built with **React**, **Redux Toolkit**, and **TailwindCSS**. Users can browse ice cream products, choose serving types (cone or cup), manage their basket, and confirm orders — all powered by a **JSON-Server** mock API.

## 🚀 Features

- 🍨 **Hero Section** – Brand messaging with trust highlights, call-to-action buttons, and a customer testimonial card with star rating.
- 🛒 **Product Catalog** – Ice cream products fetched from JSON-Server with images, prices, and descriptions.
- 🍦 **Serving Type Selection** – Choose between cone (`Külah`) or cup (`Bardakta`) before adding items to the basket.
- 📦 **Basket Management** – Add, remove, increase, and decrease quantities with Redux Toolkit state management.
- 🧾 **Basket Drawer** – Slide-out panel showing basket items, total price, and order confirmation.
- 🔔 **Toast Notifications** – Success, info, and error feedback via React Toastify for cart actions and API errors.
- ⏳ **Loading & Error States** – Skeleton placeholders while fetching data and a clear error message on failure.
- 📱 **Fully Responsive** – Warm-toned layout optimized for all screen sizes with TailwindCSS.
